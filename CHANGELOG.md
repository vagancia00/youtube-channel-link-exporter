### Added

- Portable `SKILL.md` following the Agent Skills directory convention.

- Automatic channel naming, URL canonicalization and deduplication.
- Approximate time-range filtering and incomplete-HTML protection.
- English and Mexican Spanish documentation.
- Repository artwork, tests, CI, packaging and privacy validation.
